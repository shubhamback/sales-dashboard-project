# Sales Dashboard Project (Excel)

## Overview

This project is an interactive Sales Dashboard built in Microsoft Excel to analyze sales performance, profit, orders, and quantity sold.

## Features

* KPI Cards (Total Sales, Total Profit, Total Orders, Total Quantity)
* Interactive Slicers
* Monthly Sales Trend Analysis
* Regional Sales Analysis
* Category-wise Profit Analysis
* Sales Person Performance Analysis

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting
